The CompTIA A+ Certification is awarded for demonstrating knowledge of computer hardware and software diagnostics. As well as an understanding of standard networking technologies, protocols, and standards. Those who receive this certification, must constantly update their knowledge in order to keep it current. More information available here.
C++ is a programming language used (primarily) for developing applications where efficiency is critical. Such applications involve video games and power processing. More information available here.
HTML (Hypertext Markup Language) is a markup language that is used to declare the structure of a webpage for web browsers to interpret. HTML5 being the most modern version. When used in conjunction with CSS (Cascading Style Sheets), it creates the basic building blocks of the entire internet. More information available here.
CSS (Cascading Style Sheets) is a stylesheet language that is used to define the appearance of a markup language, such as HTML. When used, it allows a programmer to describe how elements in a website should be displayed to a user. CSS3 is the most modern version. When used in conjunction with HTML, it creates the basic building blocks of the entire internet. More information available here.
JavaScript is a high-level programming language that is used in a very wide variety of applications. This is due to its simplicity and flexibility. Such flexibility allows for quicker development of video games and websites than would be possible using low-level languages. JavaScript is used in nearly all websites to provide a user interactive experience. More information available here.
jQuery is a very popular JavaScript library. This means that JavaScript developers have created a set of tools for other developers to use, that solve common problems web developers may run into. Thus, allowing web developers to spend less time reinventing the wheel and more time finding new and interesting places to use those wheels. More information available here.
Bootstrap is an HTML, CSS, and JavaScript framework. Bootstrap 4 is the latest iteration. Frameworks are similar to libraries and allow developers to focus more on their products, instead of resolving problems unnecessarily. Bootstrap focuses on a "mobile first" approach to web development, helping to ensure websites built with it are responsive and look beautiful on mobile devices, tablets, and desktop computers. More information available here.
WordPress was built with average users in mind. It allows them to create their own beautiful websites with relatively little technical knowledge. The primary advantage of this is to give users the tools they need to update their websites independent of a web developer. WordPress has a thriving community with many people creating website themes which anyone can use to make their site feel unique. More information available here.
PHP (PHP: Hypertext Preprocesser) is a server-side language used to provide resources to web browsers. Resources such as information from databases, APIs, or even just handling sensitive processing in the background. More information available here.
